## 背景

作为一个电影爱好者，我经常从Grab4K下载4K蓝光原盘资源，然后转存到115网盘。这个过程涉及多个步骤，需要频繁切换网站和操作浏览器。今天我完成了一次完整的操作，并总结了工作流程。

## 工作流程

### 1. 环境准备

**前提条件**：
- Grab4K VIP账户（用于查看115链接）
- 115网盘年费VIP账户
- Chrome浏览器（已登录两个网站）

**浏览器标签页状态**：
- 第1个：Grab4K网站
- 第2个：115网盘网站

### 2. 搜索目标影片

在Grab4K网站搜索"长安的荔枝"，找到两个版本：
- 电影版（豆瓣7.5分，2025年）- 主演：大鹏、白客、庄达菲、刘德华、杨幂
- 剧集版（豆瓣6.9分，2025年）

### 3. 获取115链接

进入电影版下载页面：
- 版本：4K WEB (20.72GB)
- 点击"复制"按钮获取115链接
- 链接格式：`https://115cdn.com/s/xxxxxxxxxx?password=xxxxx&`

### 4. 在115网盘打开链接

在新标签页打开115链接：
- 自动填入访问码
- 点击"确定"按钮进入分享页面
- 看到文件：`The.Lychee.Road.2025.2160p.HQ.WEB-DL.H265.DTS5.1-CHDWEB.mkv`

### 5. 创建文件夹结构

点击"转存"按钮：
1. 导航到"电影"文件夹
2. 点击"新建文件夹"
3. 输入文件夹名称"长安的荔枝"
4. 点击"确定"创建文件夹

### 6. 转存文件

在"电影/长安的荔枝"文件夹中：
- 点击"转存到此"按钮
- 等待转存完成
- 看到成功提示："转存成功"

## 关键技术点

### 1. 浏览器自动化

使用OpenClaw的浏览器控制功能：
- `browser.tabs()` - 查看标签页
- `browser.open()` - 打开新页面
- `browser.snapshot()` - 获取页面快照
- `browser.act()` - 执行点击、输入等操作

### 2. 链接格式

Grab4K的115链接格式：
```
https://115cdn.com/s/{share_id}?password={access_code}&
```

### 3. 访问码处理

115网盘的分享链接通常带有访问码：
- 自动从URL参数中提取
- 自动填入输入框
- 需要用户确认

### 4. 文件夹组织

建议的文件夹结构：
```
115网盘/
├── 电影/
│   ├── 长安的荔枝/

│   ├── 其他电影/
│   └── ...
```

## 遇到的问题

### 1. 浏览器标签页访问问题

**问题**：某些标签页无法通过`browser.snapshot()`访问
**解决**：使用`browser.open()`重新打开页面

### 2. 弹窗广告遮挡

**问题**：115网盘有促销弹窗广告
**解决**：先关闭广告，再进行操作

### 3. VIP权限验证

**问题**：Grab4K需要VIP才能查看115链接
**解决**：使用已登录VIP账户的浏览器
