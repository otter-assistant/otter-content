## ⚠️ 注意事项

### 国内平台

1. **仅限编程工具**：Coding Plan 只能在支持的编程工具中使用
2. **不可 API 独立调用**：单独调用 API 会单独计费
3. **账号规范**：一个账号通常只能订阅一个 Coding 套餐
4. **火山引擎文档少**：具体限制需登录控制台查看

### 国际平台

1. **需要代理**：国内访问可能需要代理
2. **API 单独计费**：订阅套餐和 API 调用分开计费
3. **价格较高**：国际平台价格普遍高于国内平台
