## 前言

写代码的时候，你是不是也遇到过这些情况？

- 敲了左括号忘记敲右括号，结果报错找半天
- 嵌套太深，看不清哪个括号配对哪个
- 想快速选中一个函数的内容，却得手动拖拽半天
- 在 JSON 里少了个引号，浏览器直接罢工

今天我要介绍一个小巧但超实用的工具 —— **BracketPlus**。它专门解决这些问题，让你的编码体验丝滑～

## 什么是 BracketPlus？

简单来说，BracketPlus 是一个专门管理"配对结构"的 IDE 插件。它的核心功能就一句话：

> 让你不再为括号、引号、标签这些成双成对的东西烦恼

它支持：
- **Brace**：`{}` 花括号
- **Bracket**：`[]` 方括号
- **Paren**：`()` 圆括号
- **Quote**：`""`、`''`、`` ` `` 引号
- **Tag**：HTML/XML 标签

## 超好用的功能

### 1. 自动补全 —— 手残党福音

你敲个 `(`，它自动帮你补全 `()`
你敲个 `{`，它自动帮你补全 `{}`
你敲个 `[`，它自动帮你补全 `[]`
你敲个 `"`，它自动帮你补全 `""`

而且它还贼聪明：
- 在字符串内部敲 `"` 不会补全
- 在注释里敲括号也不会补全
- 如果你已经手动补全了，它不会重复添加

从此告别"忘写右括号"的尴尬～

### 2. 彩色括号 —— 嵌套一目了然

```javascript
function outerFunction(param1) {
  const inner = (a, b) => {
    return { x: a, y: b };
  };
}
```

有 BracketPlus 的加持：
- 外层的 `{}` 是红色
- 内层的 `()` 是蓝色
- 最里面的 `{}` 又是红色
- 嵌套层次清清楚楚，再也不用眼瞎

### 3. 括号匹配 —— 光标跳跳跳

光标放在一个左括号上，按下快捷键（通常是 `Ctrl + Shift + \`），光标直接跳到对应的右括号！

这下再也不用：
- 一个个手动数括号
- 用眼睛找半天
- 担心是不是找错了

### 4. 快速选中 —— 一键搞定

选中一个函数、一个对象、一个数组里的所有内容？
不用手动拖拉，不用按住 Shift 敲方向键。

一键选中整个括号内容，复制粘贴超级舒服～

### 5. 自动格式化 —— 保持优雅

基于括号嵌套层级，自动帮你缩进代码。
保存文件的时候自动格式化，强迫症友好！

## 快捷键备忘

| 功能 | 快捷键 |
|------|--------|
| 跳转到匹配的括号 | `Ctrl + Shift + \` |
| 选中括号内容 | `Ctrl + Alt + ]` |
| 删除括号对 | `Alt + Shift + ]` |

（具体快捷键因 IDE 而异，以实际为准）

## 配置示例

以 VSCode 为例，在 `settings.json` 中：

```json
{
  "bracketPlus.autoClosePairs": {
    "enabled": true,
    "pairs": [
      { "open": "(", "close": ")" },
      { "open": "{", "close": "}" },
      { "open": "[", "close": "]" },
      { "open": "\"", "close": "\"" },
      { "open": "'", "close": "'" }
    ]
  },
  "bracketPlus.formatOnSave": true,
  "bracketPlus.highlightMatchingPairs": true
}
```
