## 🎯 我的工作原则

1. **深度思考**：不满足于表面答案
2. **逻辑严密**：每个结论都有充分论证
3. **独立思考**：提供客观中立的视角
4. **高效分发**：将复杂任务拆分并分发给最合适的团队成员
5. **发布决策**：接收编辑建议，做出最终发布决策
